linkdown
========

Build Webpage with Markdown, Jinja2 templates, LESS or some recent tools.

Requirements
------------

* Python
    * Python 2.7
    * jinja2
    * markdown
    * watchdog
* Nodejs
    * less
    * coffeescript

How to use
----------

1. Initialize project directory
   `linkdown init projectdir`
2. Change directory
   `cd projectdir`
3. Run server, file observer and converter
   `linkdown all -ws`
